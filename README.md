# 채팅 프로그램 구현기

## 환경 세팅

### 버전

```
# node 버전: 18.12.1
# npm 버전: 9.1.2
```

### 실행 방법

1. 프로젝트 clone 후 프로젝트 폴더 경로로 이동
   `/chatting `

2. 패키지 설치
   `npm install`

3. 서버 실행

```
npm run dev
```

### 환경 변수

```
firebase의 config

- apiKey
- authDomain
- projectId
- storageBucket
- messagingSenderId
- appId
- measurementId

```
